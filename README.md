# Portal Noticias TDI

Trabajo práctico final de Taller de Internet I.

El proyecto consiste en una página web sobre el Mundial 2026. En ella se puede encontrar información relacionada con el torneo, como noticias, estadios, grupos, países participantes y otros datos de interés.

Además, cuenta con un panel de administración al que se accede mediante un login. Desde ese panel es posible agregar, editar y eliminar noticias.

Para el desarrollo del trabajo utilizamos HTML, CSS y JavaScript. También implementamos LocalStorage para guardar las noticias creadas por el administrador y SessionStorage para mantener la sesión iniciada.

El proyecto incluye:

- Visualización de noticias del Mundial 2026.
- Panel de administración.
- Agregar, editar y eliminar noticias.
- Inicio y cierre de sesión.
- Persistencia de datos mediante LocalStorage.
- Persistencia de sesión mediante SessionStorage.
- Modo claro y modo oscuro.
. Consumo de una API para obtener la cotización del dólar.

El proyecto fue publicado utilizando GitHub Pages.
